CVlizer

CVlizer was born as a result of a job seeker's frustration. The user uploads a resume to the web platform. The parser parses all the nescessary information from the resume and compares it to attached description on the web. An automated intelligent system is required which can take out all the vital information from the unstructured resumes, extract files and store them into a huge list containing all texts in resumes which can then be ranked for a specific job postion.
